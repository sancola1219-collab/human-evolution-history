# 🦴 人類進化史 · 互動教學模擬

從 **700 萬年前**到**文明**——一個可以「**遠看**整條時間軸、**近看**每個時代寫實現場」的人類演化互動教材。純前端、零依賴、無外部圖檔，所有畫面用 Canvas 即時繪製。

## 玩法
- **遠看（時間軸）**：拖曳左右瀏覽、滾輪縮放。看見九個演化階段的姿勢與腦殼隨時間改變。
- **近看（場景）**：點任一階段，走進那個時代的寫實環境（湖畔森林、莽原、營火、冰河期、洞穴壁畫、農田…）。
  - 右側教學面板：腦容量、身高、棲地、關鍵特徵、工具、里程碑、冷知識。
  - 底部縮放滑桿：從遠景放大到解剖特寫（看腦容量比較）。
  - `← / →` 切換階段，`Esc` 回時間軸。

## 涵蓋階段
撒海爾人 → 地猿 → 南方古猿（露西）→ 巧人 → 直立人 → 海德堡人 → 尼安德塔人 → 智人 → 文明（農業革命）。

## 本機執行
```bash
node tools/server.js      # → http://localhost:8080
```
或直接用瀏覽器開 `index.html`。

## 開發者
- 架構、資料模型、紅線、地雷、部署：見 [HANDOFF.md](HANDOFF.md)。
- 改資料只動 [`data.js`](data.js)，改完跑 `node tools/validate.js`。
- 開發筆記：[`lessons/`](lessons/)。

## 技術
Vanilla JS + Canvas 2D，零依賴、零 build step。美術全部程序化向量重繪，不使用任何照片或外部圖檔。
